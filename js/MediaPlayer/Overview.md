# Overview 

**MediaPlayer** component is a HTML5 component that is used to embed a media in a web page such as web videos, YouTube, and audio files.  

## Key Features
* YouTube support
* AutoHide 
* AutoPlay
* Mute
* Shuffle
* Repeat
* Poster
* Play Speed
* Quality
* Playlist
* Title
* Full Screen
* Keyboard Navigation 
* Localization Support
